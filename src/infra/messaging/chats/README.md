# Interações

- É preciso criar uma "Linha de comunicação" com o usuário, uma espécie de "orquestrador"
  - O Orquestrador trabalha olhando a última mensagem enviada e a última recebida.
- Aceitaremos textos e áudios
- É preciso, antes de mais nada, identificar o teor da mensagem
- Uma mensagem recebida pode ser:
  - Uma resposta a um quiz
  - Uma abertura da janela de conversação (CTA)
  - Uma pergunta
  - Um pedido de tradução
